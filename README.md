# librus.ts
Library for interacting with the Librus API

## Installation
* npm:
  ```bash
  $ npm install librus.ts
  ```
* yarn:
  ```bash
  $ yarn add librus.ts
  ```
* pnpm:
  ```bash
  $ pnpm add librus.ts
  ```
